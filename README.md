\# College Fee and Student Management System (Backend)



\## Overview

The "College Fee and Student Management System" is a backend application written in Java and using Spring Boot and developed in IntelliJ IDEA. This system provides a solution for managing student data and college fee transactions.



\## Features 



\### Admin

* Perform CRUD operations on student details, fee details, miscellaneous expenses, Installment details.
* Manage student payments and track fee records efficiently.



\### Student

* View Personal information and academic details.
* Access fee, installment, and miscellaneous expense details.
* Make secure payments via eSewa.
* Login, signup and logout functionalities with proper authorization implemented.



\## Technologies Used

* Java (Backend Logic)
* Spring Boot (Framework)
* MySQL (Database)
* IntelliJ IDEA (IDE)
